## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/your-username/happinessnaylaland19.git
cd happinessnaylaland19
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## 🚀 Run Locally

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

---

## 🌐 Deploy to Vercel

1. Push project to GitHub.
2. Import repository into Vercel.
3. Add Environment Variable:

```text
GEMINI_API_KEY
```

4. Redeploy project.
